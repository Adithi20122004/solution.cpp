# GUARD LOGIC CHALLENGE - SOLUTION

## Problem Overview
This problem involves choosing the correct door between Safe Exit, Trap or return based on the given conditions.

## Approach
I used two conditions:
-If the guard is saying the truth.
-If the door marked is safe.

If the guard is truthful, the marking is trusted.
If the guard is lying, the meaning of the marking is reversed.

This is the simple function that I came up with conditional logic to know which one is the correct door.
